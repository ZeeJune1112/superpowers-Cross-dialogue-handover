# 2048 游戏 - 功能归档

## 概述
完整的 2048 益智游戏：单文件 Python/pygame-ce 应用，通过 PyInstaller 打包为独立 Windows .exe。

## 关键设计决策
- 使用 `pygame-ce`（社区版）替代 `pygame`——Python 3.14 无 pygame 预编译 wheel，pygame-ce 作为即插即用替代品，保持 `import pygame` 完全兼容
- 单文件架构：`game.py` 包含 Game2048 模型类 + pygame 渲染器 + 主循环
- 模型/视图职责分离：Game2048 类是纯逻辑（无 pygame 依赖），draw_game/main 负责渲染和交互

## 核心文件
- `game.py` — Game2048 模型 + pygame 渲染器 + 主循环（228 行）
- `test_game.py` — 20 个 pytest 测试用例，覆盖初始化、生成、移动合并、游戏结束、胜利检测
- `requirements.txt` — pygame-ce、pyinstaller、pytest
- `build.bat` — PyInstaller 一键打包脚本
- `dist/2048.exe` — 独立 Windows 可执行文件（16.4 MB）

## 测试结果
- 20/20 测试通过，连续 5 轮验证无不稳定用例
- Flaky 修复：移除 move() 后对特定单元格的 `== 0` 断言，因为 spawn() 会随机放置新 tile

## 构建输出
- `dist/2048.exe` — 无控制台窗口模式，完整可玩
